# Product Scope

## Product name

BabyShare

## Purpose

BabyShare is a web platform for sharing and selling second-hand baby items such as clothes, toys, strollers, furniture, and accessories.

The goal is to help families reuse baby items, reduce waste, and support a humanitarian initiative.

## MVP users

### Regular user

Can:
- register and log in
- create/edit profile
- post items
- browse items
- search/filter items
- view item details
- contact seller through chat
- report problematic items/users
- request listing renewal

### Admin

Can:
- view users
- view reports
- hide items
- resolve reports
- review renewal requests
- approve/reject renewals

## MVP features

- authentication
- user profiles
- item listings
- item photos
- dashboard with recent items
- search by title
- filters by category and city
- item detail page
- buyer/seller chat
- reports
- basic admin panel
- 30-day listing expiry
- manual listing renewal

## Out of scope

- automated payments
- shipping integration
- native mobile apps
- profile comments/trust reviews
- advanced filters
- automated moderation
- recommendation engine
- complex email notification system
